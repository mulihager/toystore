Toy Store

User Routes

  • POST /api/users/: Create a new user.
    Example: https://erin-courageous-bass.cyclic.cloud/api/users/

    Required Body:

    • username (string, required): The username of the user.
    • email (string, required): The email address of the user.
    • password (string, required): The password for the user account.

  • POST /api/users/login: Login with a user account.
    Example: https://erin-courageous-bass.cyclic.cloud/api/users/login

    Required Body:

    • email (string, required): The email address of the user.
    • password (string, required): The password for the user account.

Toy Routes

  • GET /api/toys/: Retrieve a list of toys.
    Example: https://erin-courageous-bass.cyclic.cloud/api/toys/

    Queries:

    • limit (number, optional): Limit the number of results per page (default: 10).
    • page (number, optional): Specify the page number for pagination (default: 1).
    • sort (string, optional): Sort the results by a field (default: "_id").
    • reverse (string, optional): Reverse the sorting order (yes/no).
    • s (string, optional): Search query to filter toys by title or info.
    • catname (string, optional): Filter toys by category name.

  • GET /api/toys/single/:id: Retrieve a single toy by ID.
    Example: https://erin-courageous-bass.cyclic.cloud/api/toys/single/123

    Parameters:

    • id (string, required): The ID of the toy to retrieve.

  • POST /api/toys/: Create a new toy.
    Example: https://erin-courageous-bass.cyclic.cloud/api/toys/

    Required Body:

    • name (string, required): The name of the toy (min: 2, max: 400 characters).
    • info (string, required): Additional information about the toy (min: 2, max: 400 characters).
    • price (number, required): The price of the toy (min: 1, max: 99999).
    • img_url (string, optional): The URL of the toy's image (min: 1, max: 10000 characters, allowed: null or empty).
    • category (string, optional): The category of the toy (min: 2, max: 400 characters, allowed: null).

    Authorization: Token required for user authentication.

  • PUT /api/toys/:id: Update an existing toy by ID.
    Example: https://erin-courageous-bass.cyclic.cloud/api/toys/123

    Parameters:

    • id (string, required): The ID of the toy to update.

    Required Body:

    • name (string, required): The name of the toy (min: 2, max: 400 characters).
    • info (string, required): Additional information about the toy (min: 2, max: 400 characters).
    • price (number, required): The price of the toy (min: 1, max: 99999).
    • img_url (string, optional): The URL of the toy's image (min: 1, max: 10000 characters, allowed: null or empty).
    • category (string, optional): The category of the toy (min: 2, max: 400 characters, allowed: null).

    Authorization: Token required for user authentication.

  • DELETE /api/toys/:delId: Delete a toy by ID.
    Example: https://erin-courageous-bass.cyclic.cloud/api/toys/123

    Parameters:

    • delId (string, required): The ID of the toy to delete.

    Authorization: Token required for user authentication.
